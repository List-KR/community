:::caution 다른 브라우저 확장 프로그램 및 애드블록 간의 충돌 주의

예상치 못한 브라우저 확장 프로그램이 안티-애드블록 팝업을 뜨게 만들 수 있습니다.
[이 목차](https://community.list-kr.com/docs/AdGuard/KnownIssues/youtube-anti-adblock#%EC%95%8C%EB%A0%A4%EC%A7%84-%EB%B8%8C%EB%9D%BC%EC%9A%B0%EC%A0%80-%ED%99%95%EC%9E%A5-%ED%94%84%EB%A1%9C%EA%B7%B8%EB%9E%A8%EB%93%A4)를 참고하셔서 알려진 브라우저 확장 프로그램들을 비활성화하시거나 삭제해주세요.

AdGuard 외 다른 애드블록[^1] 이 설치되어 있다면, YouTube 외 다른 웹 사이트에서도 충돌이 일으킬 수 있으므로, 비활성화하시거나 삭제해주세요.

[^1]: [Microsoft Edge의 추적 방지 기능 (엄격)](edge://settings/privacy)을 포함한 브라우저 빌트인 애드블록이 포함됩니다.

:::caution

웹 브라우저에 설치된 AdGuard 확장 프로그램의 필터를 업데이트하시고 **캐시, 사이트 데이터와 쿠키**를 **삭제**해주세요.
그런 뒤에 **웹 브라우저를 완전히 닫고 다시 열어주세요**.

:::note 다시 로그인해야 함

캐시, 사이트 데이터와 쿠키를 삭제하신 뒤에 웹 사이트에 다시 로그인해야 할 수 있습니다.

더욱 편리한 계정 관리를 위해 [Bitwarden](https://bitwarden.com/)[^2]이나 [KeePassXC](https://keepassxc.org/) 같은 비밀번호 매니저를 추천드립니다.

[^2]: 셀프-호스팅하신다면, [vaultwarden](https://github.com/dani-garcia/vaultwarden)이라는 옵션도 있습니다.

:::

그래도 해결히 되지 않는다면, 영어 또는 한국어[^3]로 filters@adguard.com 으로 이메일을 보내주세요.
또는, [GitHub Issue](https://github.com/AdguardTeam/AdguardFilters/issues/162262)에 영어로 공개적으로 문의를 하실 수도 있습니다.

[^3]: AdGuard 필터 유지보수자 팀에 한국어 화자가 있습니다.

## CoreLib 기반 AdGuard

AdGuard 브라우저 확장 프로그램과는 달리 CoreLib 기반 AdGuard[^4]에게는 YouTube의 안티-애드블록 개발 관심이 적어 보입니다.

[^4]: AdGuard for Windows, AdGuard for Android와 AdGuard for Mac.

[^5]: [Mozilla Firefox](about:addons); [Google Chrome](chrome://extensions/); [Microsoft Edge](edge://extensions/); [네이버 웨일](whale://extensions/)
[^6]: 유저스크립트 매니저에 설치된 YouTube 웹 사이트에서 실행되는 유저스크립트도 영향을 줄 수 있습니다.
