[microShield](https://github.com/List-KR/microShield)는 [애드쉴드 광고와 안티-애드블록 스크립트](https://www.ad-shield.io/)를 차단할 수 있는 유저스크립트입니다.

애드쉴드는 인벤 웹 사이트를 브라우징할 때 느려지는 현상과 집단적으로 애드블록을 사용하던 도중에 갑자기 광고가 차단되지 않는 현상과 관련되어 있습니다.

## 빠른 시작
이 유저스크립트를 감지시키기 위해 아래 URL를 클릭하세요.

https://cdn.jsdelivr.net/gh/List-KR/microShield@latest/microShield.user.js

### [AdGuard for iOS](https://adguard.com/ko/adguard-ios/overview.html)

유저스크립트는 현재 iOS용 AdGuard에서 지원되지 않습니다.

몰론, 일시적으로 [quoid의 userscripts](https://apps.apple.com/kr/app/userscripts/id1463298887) [^2]을 사용하실 수 있습니다. 설치 방법은 밑에 있습니다.

### [quoid의 userscripts](https://apps.apple.com/kr/app/userscripts/id1463298887) (iOS) [^2]
<details>
<summary>
    설치하기 (풀버전)
</summary>

<ReactPlayer width='560px' height='315px' controls url='https://youtu.be/MuvUlV60nq4' />

</details>


1. Apple App Store에서 quoid의 Userscript 앱을 설치해주세요.
2. 앱을 실행하여 유저스크립트가 저장될 디텍터리를 설정해주세요.
3. iOS 설정 -> Safari -> 확장 프로그램 -> Userscripts를 켜주세요.
4. iOS 설정 -> Safari -> 확장 프로그램 -> Userscripts -> 모든 웹 사이트에 대해 허용으로 바꿔주세요.
5. 아래 URL를 Safari로 열어주세요.
    ```
    https://cdn.jsdelivr.net/gh/List-KR/microShield@latest/microShield.user.js
    ```
6. Safari의 주소 막대에서 확장 프로그램 아이콘을 누르고 Userscripts 앱을 엽니다.
7.  `Tab to install`을 클릭 후 완료 버튼을 누러주세요.
8. 애드쉴드에 의해 보호받는 웹 사이트가 열려있는 탭으로 돌아가신 뒤 새로 고침 버튼을 터치해서 새로 고침해주세요.


[^1]: https://github.com/AdguardTeam/AdguardForiOS/issues/1542
[^2]: https://github.com/quoid/userscripts
